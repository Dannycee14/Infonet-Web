## 📋 About

Infonet Computers is a real Port Harcourt-based business specialising in laptop and phone sales, screen repairs, and hardware servicing. This website was designed and built from scratch to establish an online presence, showcase products, and drive customer engagement.

---

## ✨ Features

- **Multi-page layout** — Home, Services, Products, and Premium sections
- **Responsive design** — adapts across desktop and mobile screen sizes
- **Product catalogue page** — displays available laptops with descriptions
- **Services page** — details repair and support offerings
- **Custom JavaScript interactions** — dynamic UI behaviour via `script.js`
- **Node.js server** — `server.js` included for local development

## 🚀 Getting Started

### Option 1 — Open directly in browser
Just open `Infonet_main.html` in any modern browser. No build step required.

### Option 2 — Run with Node.js server

```bash
# Clone the repo
git clone https://github.com/Dannycee14/Infonet-Web.git

# Navigate into the folder
cd Infonet-Web

# Install dependencies (if any)
npm install

# Start local server
node server.js
```

Then open `http://localhost:3000` in your browser.
